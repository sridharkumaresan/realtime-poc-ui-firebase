  # realtime-poc-ui

  UI for realtime application using latest Angular and Firebase as the data source

  ### Installing

  Install required node modules.

  ```
  npm install
  ```

  ## Running the tests

  Run the unit Testing.
  ```
  npm run test
  ```

  ## Running the app locally

  ```
  npm run start
  ```
